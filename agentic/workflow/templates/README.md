# Templates

Reusable templates for workflow artifacts. **Do not edit these directly** — agents read them and create new files in the appropriate sibling folder.

| Template | Creates artifacts in | Purpose |
|----------|---------------------|---------|
| `feature-brief-template.md` | `../briefs/` | Captures problem, outcome, and scope |
| `spec-template.md` | `../specs/` | Translates brief into technical design |
| `decision-note-template.md` | `../decisions/` | Records important technical choices |
| `task-pack-template.md` | `../tasks/` | Breaks spec into implementable steps |
| `qa-report-template.md` | `../qa/` | Documents quality validation results |
