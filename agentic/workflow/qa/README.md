# QA Reports

Quality assurance reports documenting the results of all validation checks after a task pack is complete. Work is not considered done until a QA report passes.

**Naming:** `QA-NNN-<slug>.md` (e.g., `QA-001-user-auth.md`)

**Template:** [`../templates/qa-report-template.md`](../templates/qa-report-template.md)

**Created when:** All tasks in a task pack have been implemented.

**References:** Must link to its parent task pack (`TP-NNN`), spec (`SP-NNN`), and feature brief (`FB-NNN`).
