# Feature Briefs

Starting point for any new piece of work. Each brief captures the problem, desired outcome, scope, and success criteria.

**Naming:** `FB-NNN-<slug>.md` (e.g., `FB-001-user-auth.md`)

**Template:** [`../templates/feature-brief-template.md`](../templates/feature-brief-template.md)

**Created when:** A new feature, enhancement, or significant change is proposed.

**Next step:** Create a spec in `../specs/` referencing this brief.
