# Task Packs

Implementation breakdowns that divide a spec into small, focused tasks. Each task should be completable in a single agent session.

**Naming:** `TP-NNN-<slug>.md` (e.g., `TP-001-user-auth.md`)

**Template:** [`../templates/task-pack-template.md`](../templates/task-pack-template.md)

**Created when:** A spec has been approved and is ready for implementation.

**References:** Must link to its parent spec (`SP-NNN`) and feature brief (`FB-NNN`).

**Next step:** Implement tasks; open or create the matching QA report in `../qa/` early and update it incrementally; finalize when the pack is complete.
