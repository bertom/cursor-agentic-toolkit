# Decision Notes

Records of important technical or architectural decisions. Documents options considered, the choice made, rationale, and consequences.

**Naming:** `DN-NNN-<slug>.md` (e.g., `DN-001-auth-strategy.md`)

**Template:** [`../templates/decision-note-template.md`](../templates/decision-note-template.md)

**Created when:**
- Before large refactors or architecture changes
- Before introducing or replacing significant dependencies
- When multiple valid approaches exist with meaningful tradeoffs

**References:** Should link to the related spec (`SP-NNN`) if applicable.
